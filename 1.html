MINIST dataset:

The MNIST dataset plays a role in our project as it serves as a basis, for evaluating and comparing the performance of different deep learning tools. In this project we make use of the recognized MNIST dataset referred to as the "Hello World" of machine learning. Our goal is to showcase and assess the capabilities of TensorFlow, PyTorch and Flux by applying them to this dataset.

This dataset consists of 70,000 images depicting digits. It is neatly divided into a training set comprising 60,000 samples and a test set with 10,000 samples. Each image is a grayscale representation measuring 28x28 pixels. Represents digits ranging from 0 to 9. Typically, these images are stored in tensor form as matrices of size 28x28. However, they are often flattened into vectors with 784 dimensions, for processing purposes.

Our project focuses on teaching these learning models how to accurately recognize and classify these digits. By utilizing the MNIST dataset we can directly compare how each tool handles this task in machine learning, visual data classification. The simplicity and standardization of the MNIST dataset enable us to concentrate on exploring the features and user experiences provided by TensorFlow, PyTorch and Flux without being burdened by complexities associated with the data itself.

The practical involvement, in the MNIST dataset within this project will act as a reference for individuals who're new to the realm of machine learning. It serves as a starting point for tackling varied challenges in this field. Our goal with this analysis is to offer insights into which tool would be most appropriate, for different types of machine learning projects, particularly those related to image recognition tasks.
